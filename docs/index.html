TSST Language

TSST is a small programming language built in Rust. It supports variables, functions, arrays, dictionaries, loops, imports, compiled builds, packages, and simple GUI apps styled directly from TSST code.

Get Started GitHub

Quick Start

Create a workspace

tsst init MyApp
cd MyApp

Run with interpreter

tsst run main.tsst

Build to executable

tsst build main.tsst
.\build\release\main.exe

GUI apps currently run through compiled mode.

Language Basics

Variables

Use cre_ types like cre_int, cre_str, and cre_bool.

Functions

Define reusable logic with fcn. Use pub fcn main () as the entry point.

Collections

Use arrays and dictionaries with indexing, push!, and set!.

Compiled Apps

tsst build compiles TSST into Rust and creates a release executable.

Hello World

pub fcn main () {
    cons!("Hello from TSST");
}

Variables

pub fcn main () {
    cre_int age = 15;
    cre_str name = "TSST";
    cre_bool ready = true;

    cre_arr nums = [1, 2, 3];
    cre_dict user = {
        "name": "Re",
        "level": 10
    };

    cons!(name);
}

Functions

fcn add (cre_int a, cre_int b) -> int {
    return a + b;
}

pub fcn main () {
    cre_int result = add(5, 7);
    cons!(result);
}

If / Else

pub fcn main () {
    cre_int score = 90;

    if score >= 80 {
        cons!("Good");
    } else {
        cons!("Try again");
    }
}

Loops

pub fcn main () {
    for cre_int i = 0; i < 5; i = i + 1 {
        cons!(i);
    }

    cre_int x = 0;

    while x < 3 {
        cons!(x);
        x = x + 1;
    }
}

Arrays and Dictionaries

pub fcn main () {
    cre_arr nums = [1, 2, 3];
    push!(nums, 4);

    cre_dict user = {
        "name": "Re"
    };

    set!(user, "lang", "TSST");

    cons!(nums);
    cons!(user["name"]);
}

Built-ins

Built-in Use
cons!(value) Print a value.
len(value) Get string, array, or dictionary length.
input_str(prompt) Read string input.
input_int(prompt) Read integer input.
lower(str) Lowercase a string.
upper(str) Uppercase a string.
trim(str) Trim whitespace.
contains(str, needle) Check whether a string contains another string. 
pub fcn main () {
    cons!(len("hello"));
    cons!(upper("tsst"));
    cons!(contains("hello", "ell"));
}

GUI Apps

TSST can build simple GUI apps. GUI apps currently require compiled mode. The style is written in TSST using theme and layout functions.

Basic GUI

pub fcn main () {
    gui_window("My App", 600, 500);

    gui_theme_mode("light");

    gui_theme_bg(245, 245, 245);
    gui_theme_panel(255, 255, 255);
    gui_theme_card(255, 255, 255);
    gui_theme_button(255, 255, 255);
    gui_theme_button_alt(255, 255, 255);
    gui_theme_accent(0, 0, 0);
    gui_theme_text(0, 0, 0);
    gui_theme_muted(45, 45, 45);
    gui_theme_border(0, 0, 0);

    gui_style_width(420);
    gui_style_title_size(28);
    gui_style_text_size(16);
    gui_style_button_height(44);
    gui_style_spacing(10);

    gui_title("Hello GUI");
    gui_text("This app is styled from TSST.");

    gui_show();
}

pub fcn main () {
    gui_window("Supply Hustle", 620, 700);

    gui_theme_mode("light");

    gui_theme_bg(245, 245, 245);
    gui_theme_panel(255, 255, 255);

    gui_text("Reach 250 money to win.");

Current Architecture

compiler.rs
  compiles TSST AST into Rust

generated_runtime.rs
  provides generic runtime and GUI engine

main.tsst
  controls app logic, layout, and visual style

The current GUI engine is generic. App-specific logic should be written in TSST.

Roadmap

Runtime Crate

Move the generated runtime string into a real reusable Rust crate.

GUI Callbacks

Allow TSST functions to run when GUI buttons are clicked.

File APIs

Add workspace file and folder functions.

Audio APIs

Add generic audio features without per-app Rust changes.

Packages

Improve package installation and package imports.
