Overview

What is TSST?

TSST is a small language focused on readable syntax, simple types, and a clean development experience. It currently runs through a Rust-built CLI and includes a development VS Code extension for syntax coloring and snippets.

⌘

Clean Syntax

Typed declarations like cre_int, cre_str, and cre_dict.

↳

Functions

Use parameters, return values, and arrow return types like -> int.

▦

Data Structures

Arrays, dictionaries, indexing, mutation macros, and len().

✦

Editor Support

VS Code language recognition, syntax highlighting, and autocomplete snippets.

Install

Install for development

01

Clone the repo

git clone https://github.com/22aispro/TSST.git
```

cd TSST
```
02

Build the CLI

cargo build --release
03

Run a TSST file

.\target\release\tsst.exe path/to/main.tsst
Tip: Add target/release to your Windows PATH so you can run tsst main.tsst from any TSST project folder.

Built-ins

Built-ins and macros

cons!(value)

Prints a value to the console.

len(value)

Returns the length of a string, array, or dictionary.

push!(array, value)

Adds a value to an array.

set!(dict, key, value)

Sets a dictionary key to a value.

Editor

VS Code extension

TSST includes a development VS Code extension for syntax coloring, snippets, autocomplete, and .tsst file recognition. 

cd tools/vscode-tsst
```

code .
```

Press F5 in VS Code to open the Extension Development Host, then open a folder with .tsst files.
